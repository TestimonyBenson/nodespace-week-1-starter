# Week 1 Capstone: Personal Portfolio

Welcome to your first active engineering environment in the Node Space blueprint. 

## The Mission
You are tasked with building a highly responsive, ultra-fast personal portfolio. This codebase is pre-configured with Next.js 15, React 19, and Tailwind CSS. 

Your objective is to apply the concepts from Modules 1.1 through 1.4:
1. **Semantic HTML:** Ensure browsers and screen readers understand your page structure.
2. **Tailwind CSS:** Build out the UI components (Project Cards, Contact Forms) using utility-first classes.
3. **Design Systems:** Utilize the 4-point spacing grid and maintain a cohesive, striking aesthetic.

## Getting Started

1. Install your dependencies:
   ```bash
   npm install

2. Start the development engine:
```bash
npm run dev
```
3. Open http://localhost:3000 in your browser.
4. Open src/app/page.tsx and begin writing code.
